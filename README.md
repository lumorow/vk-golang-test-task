# Тестовое задание для golang dev в VK

### 1. Запуск
#### 1.1 Локальный запуск
- Склонировать репозиторий
- ```make postgresinit```: Start postgres in docker container
- ```make server```: Server
- ```make swag_ui```: Swagger

#### 1.2 Docker compose
- Склонировать репозиторий
- ```make build```: start docker containers
